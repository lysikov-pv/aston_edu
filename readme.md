## Практическое задание 5
1. Создаийте метод printThreeWords(), которыий при вызове должен отпечатать в столбец три слова: Orange, Banana, Apple.
2. Создаийте метод checkSumSign(), в теле которого объявите две int переменные a и b, и инициализируийте их любыми значениями, которыми захотите. Далее метод должен просуммировать эти переменные, и если их сумма больше или равна 0, то вывести в консоль сообщение “Сумма положительная”, в противном случае - “Сумма отрицательная”;
3. Создаийте метод printColor() в теле которого задаийте int переменную value и инициализируийте ее любым значением. Если value меньше 0 (0 включительно), то в консоль метод должен вывести сообщение “Красныий”, если лежит в пределах от 0 (0 исключительно) до 100 (100 включительно), то “Желтыий”, если больше 100 (100 исключительно) - “Зеленыий”;
4. Создаийте метод compareNumbers(), в теле которого объявите две int переменные a и b, и инициализируийте их любыми значениями, которыми захотите. Если a больше или равно b, то необходимо вывести в консоль сообщение “a >= b”, в противном случае “a < b”;
5. Напишите метод, принимающиий на вход два целых числа и проверяющиий, что их сумма лежит в пределах от 10 до 20 (включительно), если да – вернуть true, в противном случае – false.
6. Напишите метод, которому в качестве параметра передается целое число, метод должен напечатать в консоль, положительное ли число передали или отрицательное. Замечание: ноль считаем положительным числом.
7. Напишите метод, которому в качестве параметра передается целое число. Метод должен вернуть true, если число отрицательное, и вернуть false если положительное.
   Замечание: ноль считаем положительным числом.
8. Напишите метод, которому в качестве аргументов передается строка и число, метод должен отпечатать в консоль указанную строку, указанное количество раз;
9. Напишите метод, которыий определяет, является ли год високосным, и возвращает boolean (високосныий - true, не високосныий - false). Каждыий 4-ий год является високосным, кроме каждого 100-го, при этом каждыий 400-ий – високосныий.
10. Задать целочисленныий массив, состоящиий из элементов 0 и 1. Например: [ 1, 1, 0, 0, 1, 0, 1, 1, 0, 0 ]. С помощью цикла и условия заменить 0 на 1, 1 на 0;
11. Задать пустоий целочисленныий массив длиноий 100. С помощью цикла заполнить его значениями 1 2 3 4 5 6 7 8 ... 100;
12. Задать массив [ 1, 5, 3, 2, 11, 4, 5, 2, 4, 8, 9, 1 ] проийти по нему циклом, и числа меньшие 6 умножить на 2;
13. Создать квадратныий двумерныий целочисленныий массив (количество строк и столбцов одинаковое), и с помощью цикла(-ов) заполнить его диагональные элементы единицами (можно только одну из диагоналеий, если обе сложно). Определить элементы одноий из диагоналеий можно по следующему принципу: индексы таких элементов равны, то есть [0][0], [1][1], [2][2], ..., [n][n];
14. Написать метод, принимающиий на вход два аргумента: len и initialValue, и возвращающиий одномерныий массив типа int длиноий len, каждая ячеийка которого равна initialValue.
